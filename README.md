# spotify-graph
Explore music on Spotify with a graph. See it in action here: http://julienbrunuk.github.io/spotify-graph/

![demo](http://i.imgur.com/1Ra7W3z.jpg])

# What's used

* Angular
* Vis.js
* Spotify Web API
* Tooling: Bower + Sass
